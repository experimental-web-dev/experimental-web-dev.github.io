# experimental-web-dev.github.io

Portfolio/Landing Page by David A.

[Go to landing Page.](https://experimental-web-dev.github.io/)

## Commands
- Run with:
  ```console
    npm run dev
  ```
- Build page:
  ```console
    npm run build
  ```

- Deploy:
  ```console
    git gh-deploy
  ```
